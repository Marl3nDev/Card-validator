# Credit Card Validator

![Build Status](https://github.com/Marl3nDev/Card-validator/actions/workflows/deploy.yml/badge.svg)

Валидатор кредитных карт с использованием алгоритма Луна.

## Функционал
- Валидация по алгоритму Луна
- Определение платежных систем
- Форматирование номера карты
